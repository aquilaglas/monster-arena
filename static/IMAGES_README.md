# Images des Monstres

## Comment créer vos propres sprites de monstres

Pour créer des images de monstres au style rétro/pixel art, vous pouvez utiliser les outils suivants :

### Outils recommandés :
1. **Piskel** (https://www.piskelapp.com/) - Éditeur de sprites en ligne gratuit
2. **Pixilart** (https://www.pixilart.com/) - Outil de pixel art en ligne
3. **Aseprite** - Logiciel professionnel de pixel art (payant)
4. **GIMP** - Gratuit, peut être utilisé pour le pixel art

### Spécifications des images :
- **Format** : PNG avec transparence
- **Taille recommandée** : 128x128 pixels ou 256x256 pixels
- **Style** : Pixel art 8-bit ou 16-bit
- **Palette de couleurs** : Limitée (style rétro)
